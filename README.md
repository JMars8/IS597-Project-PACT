# Pact repo

The PACT project lives in the **example/** folder.

## Push to your new repository

From this folder (pact), run:

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_NEW_REPO.git
git push -u origin master
```

Replace the URL with your new repository’s clone URL. Your new repo will have everything under the **example/** folder.

## Work on the project locally

All app code is under `example/`:

- `example/frontend` – Vue UI  
- `example/backend` – Node server  
- `example/LLM` – Llama model (Python)

See **example/README.md** for how to run the app.
